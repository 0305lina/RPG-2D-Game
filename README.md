# RPG-2D-Game
This is a project to build a RPG 2D game by mainly utilizing Ptyhon, Pygame, and shelve for the basic database. 

## How to run the project

1. Install 'pip3 install pygame' through terminal
2. Run 'python3 main.py'
3. Enjoy the game!
